REQUEST PARAMS

Request paramaters are sent via an HTTP POST should be URL Encoded

Parameter Description
label Label describes the input a form creater set. Keep it unchanged from the request parameters you get on Campaign listing page "Get Lead API Parameters"
component Component describes the type of input a form creater set. Keep it unchanged from the request parameters you get on Campaign listing page "Get Lead API Parameters"
index Index is the index of field in the created form. Keep it unchanged from the request parameters you get on Campaign page "Get Lead API Parameters"
value This is the value from the end user
Response
{{{status:200,message:'Lead has been added successfully!'}| json}}

Status codes given in json response stands for -

Code Description
200 OK (Standard response for successful HTTP requests)
400 Bad Request (The server cannot or will not process the request due to an apparent client error - malformed request syntax)
401 Unauthorized
403 Forbidden (The request was a valid request, but the server is refusing to respond to it)
500 Internal Server Error (A generic error message, given when an unexpected condition was encountered and no more specific message is suitable)
